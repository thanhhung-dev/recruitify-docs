## **Lưu ý khi assign trong Jira:**

1. **Cách assign:**
   - Vào Backlog hoặc Board
   - Click vào từng issue → Assign → chọn member
   - Hoặc dùng Bulk Change để assign hàng loạt

2. **Workflow:**
   - To Do → In Progress → Done
   - Cập nhật status khi bắt đầu/kết thúc subtask

3. **Testing & Review:**
   - Người 4 (QA) review code của tất cả subtasks trước khi mark Done
   - E2E tests chỉ chạy sau khi tất cả FE/BE/DB đã hoàn thành

4. **Blockers:**
   - Task 1.1 (Turborepo) phải done trước Task 1.2-1.5
   - Task 4 phải done trước Task 5
   - Tất cả Tasks phải done trước Task 8 (E2E)
